# School Management System (SMS)

A web-based School Management System built with PHP and MySQL for managing academic and administrative activities digitally. The system supports role-based access for Admin/Teachers, Students, and Guardians.

## Features

* Role-based authentication system
* Student, Teacher, Class & Subject management
* Attendance tracking
* Exam & Result management
* Online fee payment simulation with transaction ID
* Class-wise book list management
* SQL reporting using HAVING, DISTINCT, VIEW, and IS NULL
* Secure login with password hashing and prepared statements

## Technologies Used

* PHP 7.4
* MySQL 5.7
* HTML5, CSS3, JavaScript
* Apache (XAMPP)
* GitHub 

## User Roles

* **Admin/Teacher** – Full management access
* **Student** – View attendance, results, fees, and books
* **Guardian** – Monitor child information and records 

## Installation

1. Clone the repository
2. Move the project folder to `htdocs` in XAMPP
3. Import the database into MySQL
4. Start Apache & MySQL
5. Open the project in your browser
